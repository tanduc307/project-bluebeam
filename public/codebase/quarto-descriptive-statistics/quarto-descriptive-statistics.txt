# Tình huống thường gặp

Bạn có một file dữ liệu `df` có rất nhiều hàng và cột như sau. Việc đầu tiên là ta cần làm một số lệnh để kiểm tra đặc điểm dữ liệu trước khi đi tiếp các bước phân tích sâu hơn. Để tạo ra 1 file HTML table show full dataset, bạn sử dụng package `kableExtra` với hướng dẫn ở đây[^1].

```{r, eval = FALSE}
df <- read.csv("students.csv")

library(kableExtra)
df %>% kbl(format = "html") %>%
    kable_styling(bootstrap_options = c("striped", 
                                        "hover", 
                                        "condensed", 
                                        "bordered", 
                                        "responsive")) %>%
    kable_classic(full_width = TRUE, html_font = "arial") -> output

save_kable(output, file = "df.html")
```
